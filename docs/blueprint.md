# **App Name**: Langkah Pemula Hub

## Core Features:

- Discussion Forum: Threads, comments, likes, tags, and user mentions to foster community discussion.
- Smart Event Management: Create/join events, RSVP, reminders, and event calendar to organize community activities.
- E-Course and Certification: Register for courses, access modules, complete quizzes, track user progress, and automatically generate certificates.
- Member Management: Firebase Authentication (Google + email), roles (admin, mentor, member), profile management, and activity logs.
- Digital Voting: Create polls with one vote per user and display results in real-time.
- Admin Dashboard: Administrative interface for managing users, courses, events, and forum content.
- Personalized Learning Tool: Generative AI tool to recommend relevant courses, events, and forum discussions based on user profiles and activity. The LLM will use reasoning to decide when and if to incorporate a specific piece of information to make it helpful.

## Style Guidelines:

- Background color: Dark desaturated blue (#232946) to provide a modern feel appropriate to technical content. It gives good separation in a dark-mode design.
- Primary color: Electric purple (#BE95FF) for an engaging and high-tech vibe.
- Accent color: Soft magenta (#D649D6) for highlights and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for headlines; body font: 'Inter' (sans-serif) for body text. These sans-serif fonts give a techy and modern feel. Note: currently only Google Fonts are supported.
- Use modern, outline-style icons from libraries like 'lucide-react' for a clean and consistent look.
- Employ a grid-based layout with ample whitespace, ensuring a clean and structured presentation.
- Implement subtle transitions and animations for a smooth user experience, focusing on key interactions and data updates.
